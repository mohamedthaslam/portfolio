# 🌐 Mohamed Thaslam | Developer Portfolio

A single-file developer portfolio built with a **Glassmorphism** design — featuring particle background animations and smooth scroll-reveal effects, all wrapped in a dark theme.

🔗 **Live Site:** [mohamedthaslam.github.io/portfolio](https://mohamedthaslam.github.io/portfolio/)

---

## ✨ Features

- 🎨 Glassmorphism UI — frosted-glass cards with soft blur and glow
- ✨ Animated particle background
- 🌀 Scroll-reveal animations for smooth section transitions
- 🌙 Dark theme throughout
- 📱 Fully responsive on mobile, tablet, and desktop
- ⚡ Single-file build — no frameworks, loads fast, deploys anywhere

---

## 🛠️ Built With

- HTML5
- CSS3 (Glassmorphism, keyframe animations, transitions)
- JavaScript (particle system, scroll-triggered reveals)

---

## 📂 Project Structure

```
portfolio/
├── index.html      # Main page — HTML, CSS, and JS all in one file
├── assets/         # Images, icons, resume file
└── README.md
```

---

## 🚀 Running Locally

```bash
git clone https://github.com/mohamedthaslam/portfolio.git
cd portfolio
```

Then just open `index.html` in your browser — no build step, no dependencies.
